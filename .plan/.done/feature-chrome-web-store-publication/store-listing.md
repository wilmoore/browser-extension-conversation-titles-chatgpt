## Detailed Description

### Store Description (Use this in the Chrome Web Store listing)

```
Never lose track of your ChatGPT conversations.

Conversation Titles for ChatGPT keeps your current conversation title visible at all times, making it easy to stay organized and share conversations with others.

FEATURES

• Always Visible Title
  See your conversation title displayed prominently on every ChatGPT page

• One-Click Copy
  Click the title to instantly copy it to your clipboard

• Multiple Copy Formats
  Configure shortcuts to copy exactly what you need:
  - Title only: "My Conversation"
  - Title + URL: "My Conversation https://chatgpt.com/c/..."
  - Title + Project: "[Project Name] My Conversation"
  - Markdown link: "[My Conversation](https://chatgpt.com/c/...)"

• Customizable Shortcuts
  Use Click, Shift+Click, Ctrl+Click, or Ctrl+Shift+Click for different formats

• Optional Audio Feedback
  Hear a subtle confirmation sound when you copy (can be disabled)

PRIVACY FOCUSED

• No data collection
• No external servers
• No tracking or analytics
• Works entirely offline
• Open source

PERFECT FOR

• Researchers managing multiple AI conversations
• Developers documenting ChatGPT-assisted coding sessions
• Writers organizing brainstorming sessions
• Teams sharing conversation references
• Anyone who uses ChatGPT regularly

HOW TO USE

1. Install the extension
2. Visit any ChatGPT conversation
3. See your title displayed at the top of the page
4. Click to copy in your preferred format
5. Customize shortcuts via extension options

SUPPORTED SITES
• chatgpt.com

Questions or feedback? Contact support@savvyai.dev

Made with care by SavvyAI
```
